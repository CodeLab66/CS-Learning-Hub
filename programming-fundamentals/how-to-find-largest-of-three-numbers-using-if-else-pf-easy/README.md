# How to Find the Largest of Three Numbers Using If-Else in Python, JavaScript and C++

## Problem Statement
Write a program that takes three numbers as input from the user and prints the largest among them.

## Explanation
This problem helps beginners understand how to compare multiple values using conditional statements.

When comparing three numbers, we must check which number is greater than the other two. This requires combining logical conditions using the AND operator.

By solving this problem, you will learn:
- How to compare multiple numbers
- How to use logical operators (AND)
- How to structure multiple conditional checks

This type of comparison logic is commonly used in grading systems, ranking systems, and decision-making applications.

## Approach
1. Take three numbers as input from the user
2. Compare the first number with the other two
3. If it is greater than both, print it
4. Otherwise compare the second number with the other two
5. If none of the above, the third number is the largest

## Sample Input
10 25 15

## Sample Output
25
