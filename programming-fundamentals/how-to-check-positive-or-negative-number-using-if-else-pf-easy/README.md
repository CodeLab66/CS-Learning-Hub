# How to Check Positive or Negative Number Using If-Else in Python, JavaScript and C++

## Problem Statement
Write a program that takes an integer as input from the user and checks whether the number is positive, negative, or zero.

## Explanation
This problem helps beginners understand how to use conditional statements to make decisions in a program.

A number can be:
- Positive (greater than 0)
- Negative (less than 0)
- Zero (equal to 0)

The program should take input from the user and use if-else conditions to determine the type of number.

By solving this problem, you will learn:
- How to compare numbers
- How to use multiple conditions in if-else
- How to handle different logical cases

Conditional checks like this are commonly used in validation systems, financial applications, and data analysis programs.

## Approach
1. Take an integer input from the user
2. If the number is greater than 0, print "Positive"
3. If the number is less than 0, print "Negative"
4. Otherwise, print "Zero"

## Sample Input
-5

## Sample Output
Negative
