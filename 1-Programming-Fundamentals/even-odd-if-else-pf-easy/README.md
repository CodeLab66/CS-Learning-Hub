# Even or Odd Number

## Problem Statement
Write a program that takes an integer as input from the user and checks whether the number is even or odd.

## Explanation
This problem helps beginners understand how decision-making works in programming using conditional statements.

A number is considered even if it is completely divisible by 2.  
If the number is not divisible by 2, then it is considered odd.

In this problem, the program takes an integer input from the user and checks whether the remainder when divided by 2 is equal to zero or not.

By solving this problem, you will learn how to:
- Use conditional statements (if-else)
- Perform basic mathematical operations
- Make decisions in a program based on conditions

Conditional statements are widely used in real-world applications such as login validation, grading systems, and access control systems.

## Approach
To solve this problem, follow these steps:

1. Ask the user to enter a number
2. Check if the number is divisible by 2
3. If the remainder is 0, print "Even"
4. Otherwise, print "Odd"

## Sample Input
4

## Sample Output
Even
