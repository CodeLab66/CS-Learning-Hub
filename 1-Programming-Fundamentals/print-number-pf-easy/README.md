# Print a Number
## Problem Statement
Write a program that takes an integer as input from the user and prints the same number on the screen.

## Explanation
This is one of the first problems that beginners solve while learning programming.  
The main goal of this problem is to understand how a program can interact with a user.

In this problem, the user enters a number using the keyboard. The program then stores this number inside a variable and displays the same number as output on the screen.

By solving this problem, you will learn how to:
- Take input from the user
- Store input in a variable
- Display output on the screen

These steps are used in almost every real-world program such as login systems, registration forms, calculators, and data entry applications.

## Approach
To solve this problem, follow these steps:

1. Ask the user to enter a number
2. Store the entered value in a variable
3. Print the stored value on the screen

## Sample Input
5

## Sample Output
5
